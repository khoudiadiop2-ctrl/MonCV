# CV Interactif - Khoudia Diop

## Description

Ce projet consiste en la réalisation d'un CV en ligne interactif, conçu dans le cadre du module de Développement Web (Licence 2 Informatique).

## Technologies utilisées

- HTML5 (Sémantique)
- CSS3 (Responsive, Flexbox/Grid)
- JavaScript (Vanilla JS)

## Fonctionnalités

- Design responsive (Adaptation mobile)
- Mode sombre / clair avec sauvegarde locale
- Menu hamburger pour mobile
- Formulaire de contact avec validation JS
- Animation des compétences au scroll
- Bouton "Retour en haut"

## Instructions

1. Ouvrez le fichier `index.html` dans votre navigateur.
2. Pour tester le mode sombre, cliquez sur le bouton dédié.
3. Utilisez le menu hamburger pour naviguer sur mobile.
